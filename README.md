# SCP-035
This plugin adds SCP-035 to the game, SCP-035 will appear as a Tutorial and glow red. The mask item appears as a glowing red SCP-268, when it is used, the player turns into SCP-035. When SCP-035 dies, there's a chance for the mask item dropping. Everything (Almost) is configurable in the config.

*Requires Harmony0.dll*
